Symbolic calculations::

    sage: var('x,y')
    (x, y)
    sage: f = sin(x) - cos(x*y) + 1 / (x^3+1)
    sage: f
    1/(x^3 + 1) - cos(x*y) + sin(x)
    sage: f.integrate(x)
    1/3*sqrt(3)*arctan(1/3*sqrt(3)*(2*x - 1)) ...

::

    sage: expr = sin(x) + sin(2 * x) + sin(3 * x)
    sage: solve(expr, x)
    [sin(3*x) == -sin(2*x) - sin(x)]

.. No solution!
.. Numeric solution:

::

    sage: find_root(expr, 0.1, pi)
    2.0943951023931957
